# Automating-Port-Operations-using-Deep-Learning-and-Transfer-Learning
Developed a deep learning model to classify boats for automated reporting. Built a custom CNN and applied transfer learning using MobileNetV2 to create a lightweight, efficient model suitable for mobile deployment, enabling accurate and scalable real-time predictions.


### 📌 Project Overview

Marina Pier aims to automate boat recognition using deep learning to streamline reporting systems. This project focuses on building an efficient image classification model capable of identifying different types of boats, with an additional goal of deploying the solution on mobile devices.

---

### 🎯 Objectives

* Develop a **Convolutional Neural Network (CNN)** for boat classification
* Implement **transfer learning** using a lightweight pre-trained model
* Optimize the model for **mobile deployment**

---

### 🧠 Approach

#### 1. CNN Model Development

* Built a custom CNN architecture for image classification
* Applied preprocessing and data augmentation techniques
* Trained and evaluated model performance

#### 2. Transfer Learning (Mobile Optimization)

* Used MobileNetV2 as a lightweight backbone
* Fine-tuned layers to improve classification accuracy
* Reduced model size and computational cost for mobile deployment
